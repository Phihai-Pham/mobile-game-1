# Simple Tower Defense Game
A beginner-friendly 2D Tower Defense game made with Unity for Android.



## Features
\- Enemy spawning system

\- Towers automatically attack enemies

\- Bullet and damage system

\- Enemy health

\- Simple wave gameplay

\- Mobile touch support

\- Android build support



## Built With
\- Unity

\- C#

\- Unity 2D Tools



## Project Structure
Assets/

├── Scenes/

├── Scripts/

├── Prefabs/

├── Sprites/

└── UI/



## Scripts

### Enemy.cs

Handles:

\- Enemy movement

\- Health system

\- Damage handling



\### Tower.cs

Handles:

\- Enemy detection

\- Shooting bullets

\- Attack range



### Bullet.cs

Handles:

\- Bullet movement

\- Damage dealing

\- Target tracking



### Spawner.cs

Handles:

\- Enemy spawning

\- Spawn timing



### GameManager.cs

Handles:

\- Player money

\- Lives

\- Global game state



## How to Play

1\. Place towers on tower spots

2\. Towers automatically attack enemies

3\. Prevent enemies from reaching the target

4\. Survive as many waves as possible



## Controls

### Android

\- Tap to place towers

\- Tap UI buttons



### PC

\- Mouse click interaction
